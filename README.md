# AceInterview

> Ace Every Interview with AI

## 📖 About

AceInterview is an AI-Powered Interview Preparation Platform that helps students prepare for technical interviews through coding practice, MCQs, AI-powered mock interviews, resume analysis, and performance tracking.

## 🚀 Tech Stack

### Frontend
- React
- Tailwind CSS
- React Router
- Axios

### Backend
- Java
- Spring Boot
- Spring Security
- JWT

### Database
- MySQL
